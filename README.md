# MonatsAbschluss1

In diesem Projekt haben wir Apotheken Verkaufsdaten simuliert. 

---

## 📊 Projektübersicht

* **Datengenerierung:** Die simulierten Rohdaten wurden mittels Python generiert und in der Datei [`apotheken_verkaeufe.csv`](./apotheken_verkaeufe.csv) exportiert.
* **Datenanalyse & Dashboard:** Die Daten wurden über Power Query in die Excel-Datei [`Apotheken_Analyse.xlsx`](./Apotheken_Analyse.xlsx) geladen. Beim Einlesen wurde PowerQuery genutzt, um die Trennzeichen der Gleitkommazahlen sauber zu interpretieren. Dort befindet sich ein Dashboard, das folgende Aspekte untersucht:
  * 🏷️ **Rabatt-Analyse:** Untersuchung der Verkäufe mit vs. ohne Rabatt.
  * 💰 **Umsatz-Erlöse:** Detaillierte Aufschlüsselung der Einnahmen pro Medikament.
  * 📋 **Rezeptpflicht-Struktur:** Unterscheidung nach verschreibungspflichtigen und frei verkäuflichen (OTC) Medikamenten.
